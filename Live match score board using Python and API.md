# 🏏 Live Match Scoreboard

A Python-based project that fetches and displays live cricket match scores using an external API.

---

## 🚀 Features
- Fetches real-time match data
- Displays team names, score, and match status
- Simple and clean terminal output
- API integration using Python

---

## 🛠️ Tech Stack
- Python
- Requests Library
- REST API
